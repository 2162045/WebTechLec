What is URL?

Uniform Resource Locator or URL, is the name of the website itself. It is a way for identifying the location of web resources over the Internet. URL is also called as a Web Address. Examples of URLs are: *http://www.youtube.com *http://www.google.com

Structure of URL: *https://www.lifewire.com/what-is-a-url-2626035 HTTPS: This stands for Hypertext Transfer Protocol Secure. This is the language that your browser and computer used to understand each other. WWW:The World Wide Web is a combination of resources located on the Internet. DOMAIN NAME: This is simple the name of the site where users can access it easily. Domain names have numerical equivalents (IP address - e.g. 192.168.1.11). Each domain name is unique. EXTENSIONS: These are top-level domains that gives you information what the site is about or where it is based from.
