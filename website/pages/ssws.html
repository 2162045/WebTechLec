Server Side Web Scripting

JAVA Servlets(HTTP) Hosted in a servlet container

Java servlets are java objects that are intended to play the role of server components in a client-server communications.

Servlet Lifecycle instantiation

constructor call - creates an instance of the servlet of service client requests.

initialization

init method - invoked only once and is intended for any startup initialization code for the servlet.

request handling

service method - servlet performs whatever logic necessary in order to serve the request and generate response.

destination destruction(house cleaning)

destroy method - invoked before the servlet instanceis unloaded.

HTTP Servlet container (Web container)

Java servlet that is used to handle HTTP requests and HTTP responses. This is hosted in a web container.
