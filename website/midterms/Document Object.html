DOM

Document Object Model

DOM stands for Document Object Model is a W3C standard that defines the standards for accessing documents.It is a programming API for XML and HTML documents that defines how the document should be accessed and manipulated.It was originated for the allowment of the portability among web browsers of the JavaScript scripts and Java programs. W3C separated DOM standard into 3 different parts. The standard model for all the document type which is the Core DOM, then the XML DOM for the XML documents and the standard model for HTML Document.
