Properties

  • Shorthand properties

    • It allows specify the values of several properties with single property.

  • Vendor specific extensions (vendor prefixes)

    • It specifies the non-standard or experimental.
    -webkit- , -moz- , -o- , -ms- , -etc- used by browsers vendors as prefix for the names of experimental or non standard

  • Custom properties a.k.a CSS variables (experimental)

    • property named prefixed with --,representing a value that can be used throughout the document using var function.
