Event Handler Mapping (DOM)

Event is a signal that an action occurs. DOM nodes are the one who creates signals

Some events are mouse events, form elements events, keyboard events, document events and css events. Event handler is a program script that triggers if an action occur.

Ways to assign Event Handler

  • HTML attribute
  • DOM property
  • Methods

There are two phase:

    For example you trigger the element

  • Capture Phase - starts in the handler, then to its parents, then follows all the way up to its ancestors.
  • Bubble Phase - starts in its parent, then follows its children until it reach the handler'.

Event Handler Registration

There are 4 ways of implementing

  • Method 1 - Inline HTML event attribute
  • Method 2 (pre-DOM level2)

    • Can register more than one element

  • Method 3 DOM L2 - event registration

    • Can register more than one element

    can register handler on bubble phase and capture phase

  • Method 4 (Microsoft-specific, non standard)

    • Can register more than one element

    can register handler only on bubble phase
