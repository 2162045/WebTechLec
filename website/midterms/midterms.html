CSS Declarations

*Properties

-Shorthand properties

These are properties that allows you to set values of numerous CSS properties. Usign such property will save you time and can wrote more short and to the point stylesheets.


BACKGROUND PROPERTIES: This is a shorthand property that defines values of either a background's image or color.

EXAMPLES:


*background-attachment sets whether the background image if fixed or an image scrolls.


*background-blend-mode sets the blending of certain background layers.


*background-clip defines how far should the background extend inside an element.


*background-color sets the color of an element.


*background-image sets multiple background images for a specific element.


-Vendor specific extension

These extension start with either a dash ("-") or an underscore ("_"), and then it is followed by an abbreviation of a specific browser extension. An example is "-webkit-" for WebKit based browsers. Thses prefixes are then followed by a property name. Examples are "-webkit-border-radius" and "-moz-border-radius".


Common Vendor specific extensions that you should be familiar with are:

Webkit-based browsers (as mentioned earlier), like Chrome, which uses "-webkit-" tag.


Gecko-based browsers, like Firefox, which uses "-moz-" tag.


Proprietary Internet Explorer extension that uses "-khmtl-" tag.
