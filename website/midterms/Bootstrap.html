Bootstrap

Boostrap is a toolkit for designing and developing responsive web applications. It is free, open source popular front-end web component framework that includes HTML, CSS and JavaScript. Twitter Blueprint was the original name of bootstrap it was developed by Mark Otto and Jacob Thornton at Twitter.

Bootstrap helps the developer to create and develop web applications in more easy, simple and faster ways. By using templates and plugins you can now create and customize your own site
