Safe Methods - read only semantics (GET, HEAD, OPTION, TRACE) Idempotent Methods - multiple identical request (GET, HEAD, OPTION, TRACE, PUT, DELETE) Cacheable Methods - response to method that allows to be stored with feature reuse. (GET, HEAD, POST)
