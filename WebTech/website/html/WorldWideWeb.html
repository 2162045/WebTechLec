The World Wide Web

The World Wide Web was invented in 1989 by Sir Tim Berners-Lee, a software engineer at CERN, the WWW allows the exchange of information of different users through HTTP.

The three foundations of the web developed by Tim Berners-Lee

HTML(HyperText Markup Language) the standard markup language in creating and defining the structure of web pages

Tim Berners-Lee created the first webpage browser, “WorldWideWeb.app” and the first server “httpd” by 1990 the page was released, and by 1991 it was made public.

On April 1993, CERN announced that the web would be free for the public without any fees. By 1994 Sir Tim moved to the Massachusetts Institute of Technology and founded the World Wide Web Consortium (W3C) an international community dedicated to develop open web standards. W3C has defined standards that will allow web developers to build rich and interactive web applications.
