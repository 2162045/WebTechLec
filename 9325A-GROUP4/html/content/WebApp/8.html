A8:2017-Insecure Deserialization

According to OWASP “Exploitation of deserialization is somewhat difficult, as off the shelf exploits rarely work without changes or tweaks to the underlying exploit code. ” It occurs when data is used to abuse the logic of an application and this data are untrusted one. To prevent this vulnerability you should know how to identify the possible holes of the website and how to resolves it. And rejects untrusted sources.
