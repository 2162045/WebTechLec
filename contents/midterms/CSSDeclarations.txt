    <p>-Shorthand properties</p>
            <p>These are properties that allows you to set values of numerous CSS properties. Usign such property will save you time and can wrote more short and to the point stylesheets.</p>
            <br>
            <p>BACKGROUND PROPERTIES: This is a shorthand property that defines values of either a background's image or color.</p>
            <p>EXAMPLES:</p>
            <br>
            <p>*background-attachment sets whether the background image if fixed or an image scrolls.</p>
            <br>
            <p>*background-blend-mode sets the blending of certain background layers.</p>
            <br>
            <p>*background-clip defines how far should the background extend inside an element.</p>
            <br>
            <p>*background-color sets the color of an element.</p>
            <br>
            <p>*background-image sets multiple background images for a specific element.</p>
